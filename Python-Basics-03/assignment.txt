Q1. Palindrome Checker
Ask the user for a string and check whether it is a palindrome or not.

A palindrome is a string which is the same when we read it forward & backward.
Eg - "madam", "racecar" etc.
A palindrome string is equal to the reversed version of the string. We can use a loop to reverse the string manually.

[Use sets]

Examples:

python

# Share common elements
list1 = [1, 2, 3]
