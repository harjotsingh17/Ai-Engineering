# Variables in Python

A variable is a container that stores data.

### Examples

name = "Harjot"
age = 30
PI = 3.14

print("Value of PI =", PI)

---

## Variable Naming Rules (Identifiers)

1. Must start with a letter or underscore
2. Can contain letters, numbers, underscores
3. Cannot use Python keywords (if, for, class, etc.)
4. Python is case-sensitive (age and Age are different)

### Invalid Examples

2name = "Bob"
class = 10
