# Data Types in Python

A data type defines the type of value a variable holds.

| Data Type | Example | Description |
|-----------|---------|-------------|
| int | 10 | Whole numbers |
| float | 3.14 | Decimal numbers |
| str | "hello" | Text data |
| bool | True | True or False |
| None | None | No value |

### Checking Data Type

x = 10
print(type(x))
